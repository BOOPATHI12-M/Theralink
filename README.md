# TheraLink - IoT Assistive System

TheraLink is an IoT-based assistive technology project developed to support people affected by stroke and severe physical disabilities. The system enables patients to interact with and control devices independently using multiple accessible input methods.

## Features
- Voice Control - Operate devices using voice commands
- Gesture/Finger Control - Control system through finger movements
- Breath Control - Access system using breathing signals for users with limited mobility

## Technologies Used
- ESP32
- Arduino Programming
- IoT Communication (WiFi + HTTP)
- Sensors and Relay Modules
- Embedded Systems
